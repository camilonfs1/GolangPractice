# FMT

## General 

+ %v (value in default format)
+ %T (type)
+ %% (literal %)

## Boolean

+ %t (true or false)

## Integer

+ %b (base2)
+ %o (base8)
+ %d (base10)
+ %x (base16)
+ %X (base16) capital letter

## Floating point

* %e (scientific notation)
* %f / %F (decimal no exponent)
* %g (for large exponents)
